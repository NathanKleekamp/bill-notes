# In Force RFCs

## Use SingleSPA Parcel to wrap all remote modules

- Link to the [RFC](https://billcom.atlassian.net/wiki/spaces/ACP/pages/3021898231/Use+SingleSPA+Parcel+to+wrap+all+remote+modules)

Outcome:

- All exposed remote modules should be built utilizing a [single-spa
  Parcel](https://single-spa.js.org/docs/parcels-overview) adapter

- Host modules must consume remote modules with their frameworks respective
  single-spa adapter
